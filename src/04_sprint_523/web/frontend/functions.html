How to Use This Website

A guide to all the features and tools available in the Fraud Corpus interface.

Corpus Search

📂

Corpus Selection

Use the left sidebar on the Search page to switch between different corpora.

  • English – Search only English-language documents
  • Chinese – Search only Chinese-language documents
  • All – Search across both corpora simultaneously
📊

Statistics

View detailed statistics and visualizations about the corpus data.

  • Filter statistics by corpus type: English, Chinese, Annotated, or All
  • View total document counts, annotated vs. raw distribution
  • Interactive charts with selectable chart types (Doughnut, Bar, Pie, Polar Area)
  • Detailed label breakdown table with percentage bars
🛠️

Query Builder

Build advanced search queries without memorizing query syntax.

  • Combine multiple search conditions with AND / OR operators
  • Filter by metadata fields such as label type
  • Access it through the "Query Builder" link on the Search page
🏷️

Metadata Filters

Narrow your search results by document metadata on the right sidebar.

  • Ham / Spam / Phishing – Filter by email classification label
  • Concordance View – See search terms in context with surrounding text
  • Frequency List – View word frequency distributions for your query
ℹ️
